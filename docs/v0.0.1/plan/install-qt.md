# Install Qt

- Docmentation for installing Qt on Linux can be found at the following link:
> https://doc.qt.io/archives/qt-5.15/linux.html

- Online installation of Qt can be done using the following command:

> https://www.qt.io/development/download-open-source

```bash
sudo apt-get install build-essential libgl1-mesa-dev

sudo chmod +x qt-online-installer-linux-x64-*.run

sudo ./qt-online-installer-linux-x64-*.run
```
